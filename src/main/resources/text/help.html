Help in Hanne Sergine

Here are answers to a few common questions about Hanne Sergine:
  • What does Hanne Sergine do?
      • the user to select one or more specific edits in any order. The selected edits can be undone, deleted or clustered together to form a group.
      Users also have the ability to select multiple edits spread over several different groups and undo or delete them in one step.
      Undone edits and groups can be recovered. However, deleted edits are permanently gone.
  • How can I run this application?
      • To open Hanne Sergine, select the Start button and then type Hanne Sergine in the search box. Select Hanne Sergine from the results.
  • How do I use Hanne Sergine to edite document?
      • If you run Hanne Sergine application and go to the "Edit" tab you can manage your edits.
