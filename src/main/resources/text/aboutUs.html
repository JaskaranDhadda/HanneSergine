About Hanne Sergine

    Hanne Sergine is a simple text editor, coded in Java, that allows users to manage their edits history more efficiently than traditional undo options.
    Indeed, the user's modifications can be clustered together into different groups and the user has the ability to delete any specific edits in any order or entire groups of edits.
    In addition, Allows the user to select one or more specific edits in any order. The selected edits can be undone, deleted or clustered together to form a group.
    Users also have the ability to select multiple edits spread over several different groups and undo or delete them in one step.
    Undone edits and groups can be recovered. However, deleted edits are permanently gone.
